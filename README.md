# BookMarker Application 🔖
#### A simple and efficient web application that allows users to save and manage their favorite website bookmarks locally.

## 🚀 Features
#### Add Bookmarks: Save a website name and its URL.

### Local Storage Persistence: Your bookmarks stay saved even after refreshing the page.

### Visit Sites: Quickly open your saved bookmarks in a new tab.

### Delete: Remove bookmarks you no longer need with a single click.

### Validation: Ensures valid URLs are entered before saving.

## 🛠️ Tech Stack
#### HTML5 & CSS3 - For the structure and styling.

#### JavaScript (ES6+) - For logic and Local Storage management.

#### Bootstrap - For responsive design.
